Installation
============

Using pip
---------

.. code-block:: bash

    pip install od-draw

From source
-----------

.. code-block:: bash

    git clone https://github.com/ajcarter/od-draw.git
    cd od-draw
    pip install -e .

Development installation
------------------------

.. code-block:: bash

    git clone https://github.com/ajcarter/od-draw.git
    cd od-draw
    pip install -e ".[dev,docs]"
